Mongodb Connection Code

Personal Financial Assistant is more like an interface rather than a typical web application. It do not have its database behind. But surprisingly that is why PFA is powerful! As long as user provide a mongodb connection code, PFA can instantly connect to that mongodb and work for the user. MongoDB Atlas does provide free online database service and general user can apply for a free account from them. For personal usage, the free account is more than enough for PFA but if you want, you can buy an advanced account from mongodb altas


If PFA is built on its own database, it will not be able to handle endless user requests. But now it act as an interface between you and your own database. One user, one database, not network congestion, not database space limit. With PFA, you are always enjoying the best of the best!

Instruction on getting you code

Get Your online database

Go to Mongodb Altas, and follow their instruction to get your own online database

Get Your Code for free

Add a user with read and write priority

Remember to allow all ip address in IP Whitelist

Get your code

At Cluster, click connect-->Connect Your Application-->I am using driver 3.6 or later. Copy the SRV address and replace the password with your own password. That will be your Mongodb Connection Code

Database Encryption Password

Mongodb Altas is public, it is possible that someone else may somehow break into your database and read everything their although the probability is very very low. But with Database Encryption Password, you won't need to worry about that since all your data in the database are encrypted by this code. We do not store the code for you so remember it by yourself. We believe this is the safest way to protect your personal privacy. Just input any password you like and can remember, and keep using that

What if you forget your Password

Sorry for that, we can't help you. But you can use a new password. Every time, you can login with any password you want and only those data encrypted by this password will be presented to you! That means, you can share the Mongodb Connection Code with your family and keep your own Database Encryption Password. (Although we encourage you to use a fixed Database Encryption Password since to test which data is encrypted by your current password cost time)

No correct or wrong

As explained above, you will find you can access to the service with any Database Encryption Password. If you find your data is not there after login, it is likely that you typed in a different password. You can by yourself, to maintain two different password for different purpose. Under that case, you will see two set of data.

Login Failed

If you failed to login, you will by default connect to a public demo database. The login button will still be there for you to login

The only reason for login failure is an incorrect Mongodb Connection Code, check it please, if you can connect to your database with other application like mongodb compass, you should be able to connect to it with PFA
